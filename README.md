# Modelos de Avaliação de Impacto

Este repositório tem como objetivo servir como **material de apoio para revisão de métodos de avaliação de impacto**.  

Os códigos foram organizados com fins **didáticos**, buscando facilitar a compreensão e a replicação de diferentes métodos econométricos utilizados na avaliação de políticas públicas.

## Sobre os códigos

O conjunto de códigos presentes neste repositório é uma **combinação de implementações baseadas em diferentes bibliotecas, autores e materiais de referência**.  

Sempre que trechos de código forem baseados diretamente no trabalho de outro autor, a **referência correspondente será citada**.

Os códigos estão implementados em **R e Python**, dependendo do método e das bibliotecas mais adequadas para cada abordagem.

Em alguns casos, para facilitar a interpretação e a compreensão dos métodos, foram utilizados **trechos literais das referências consultadas**, sempre com a devida indicação da fonte.

## Referências

Para cada método apresentado, foram utilizadas **referências de livros e artigos acadêmicos**.  
Sempre que possível, foram priorizadas **referências mais simples e acessíveis**, com o objetivo de facilitar o processo de aprendizado e revisão dos tópicos.

## Métodos abordados

Os principais métodos de avaliação de impacto revisados neste repositório são:

1. Aleatorização  
2. Respostas binárias  
3. Dados em painel  
4. Variáveis instrumentais  
5. Regressão quantílica  
6. Pareamento (Matching)  
7. Diferenças-em-Diferenças (Diff-in-Diff)  
8. Regressão descontínua  
9. Controle sintético  

## Objetivo do repositório

Este repositório **não tem como objetivo apresentar implementações originais**, mas sim **organizar exemplos de códigos e referências** que possam auxiliar na revisão e no aprendizado dos principais métodos utilizados em avaliação de impacto.
